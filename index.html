  • DIY software For Modelling and Simulation 6-DOF Manipulation Robot

    This project was a part of my Final Thesis in Robotics course in University, my team was tasked with automating the DIY 6-DOF Manipulation Robot to pick and place sets of objects in a workspace. This was done using the PyQt5 module of Python scripting platform to create a controlling entire movement of robot model, this controller system be used to publish commands by means of the intrinsic interface as well as MoveIt! software, or publish commands by voice command systems with which we were able to build a virtual representation of the environment and sensor information during operation.

    The observable process can be broken down into several iterative steps:

    1. Return to Home position
    2. Acquire position and orientation of nearest object
    3. Pick object accordingly
    4. Place object in designated zone, stacked atop one another. conducted on simulation
    5. conducted an automated movement by voice command

    documentation can be found on Github here. The robot is able to consistently sort several types of objects, account for alterations to the workspace between cycles
